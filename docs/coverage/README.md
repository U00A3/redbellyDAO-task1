# Test coverage artifact

Generated by `npm run coverage` (Hardhat + solidity-coverage).

| File | Description |
|------|-------------|
| [`coverage-final.json`](coverage-final.json) | Istanbul coverage report (line/branch/statement data per contract) |
| [`summary.json`](summary.json) | Coverage summary JSON from the same run |

**Last run result:** 100% line coverage on all contracts (16 tests passing).

Regenerate locally:

```bash
npm install
npm run coverage
cp coverage/coverage-final.json docs/coverage/
cp coverage.json docs/coverage/summary.json
```
